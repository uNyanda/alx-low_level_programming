# Preprocessor

## Tasks

**0. Object-like Macro**

- [x] Create a header file that defines a macro named `SIZE` as an abbreviation for the `1024`.

- :file_folder: : `0-object_like_macro.h`


**1. Pi**

- [x] Create a header file that defines a macro named `PI` as an abbreviation for the token `3.14159265359`.

- :file_folder: : `1-pi.h`


**2. File name**

- [x] Write a program that prints the name of the file that was compiled from, followed by a new line.
- You are allowed to use standard library

- :file_folder: : `2-main.c`


**3. Function-like macro**

- [x] Write a function-like macro `ABS(x)` that computes the absolute value of a number `x`.

- :file_folder: : `3-function_like_macro.h`


**4. SUM**

- [x] Write a function-like macro `SUM(x, y)` that computes the sum of the numbers `x` and `y`.

- :file_folder: : `4-sum.h`
